Alambana Chatbot
🔑 Cara Mendapatkan OpenRouter API Key:

1. Kunjungi: https://openrouter.ai/keys
2. Sign Up/Login dengan akun Google atau GitHub
3. Klik "Create Key" untuk membuat API key baru
4. Copy API Key yang dimulai dengan "sk-"
5. Paste di atas dan pilih model, lalu mulai chat!

💰 Model Gratis: DeepSeek, Llama, Gemma, Zephyr (tanpa biaya)
💳 Model Premium: GPT, Claude (berbayar sesuai pemakaian)

✨ API key Anda aman dan hanya digunakan untuk chat ini!
